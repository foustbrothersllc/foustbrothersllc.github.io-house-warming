# Housewarming Flyer — live, clickable invite

A single self-contained page. The address opens Google Maps, the phone number taps to call/RSVP, and the registry button opens your Walmart wishlist. Nothing else to install.

## Put it online with GitHub Pages (free, ~3 minutes)

1. Go to https://github.com/new and create a repository.
   - Name it something like `housewarming` (any name is fine).
   - Set it to **Public**.
   - Click **Create repository**.
2. On the new repo page, click **uploading an existing file**.
3. Drag in **`index.html`** (the file in this folder). Click **Commit changes**.
4. Go to the repo's **Settings → Pages** (left sidebar).
5. Under **Build and deployment → Source**, pick **Deploy from a branch**.
   Set **Branch** to `main` and folder to `/ (root)`. Click **Save**.
6. Wait ~1 minute, then refresh. GitHub shows your live link at the top of the Pages screen:

   `https://YOUR-USERNAME.github.io/housewarming/`

That URL is what you text or post. On a phone, all three buttons work.

## Updating later
To swap the photo or change details, replace `index.html` with a new version
(commit over it) and the live link updates automatically within a minute.
